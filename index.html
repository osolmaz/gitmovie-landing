GitMovie: Develop videos like software

Imagine that YouTube and GitHub had a baby.

The idea is simple enough:

  • Develop entire videos by programming, without touching a video editor.
  • Collaborate with others on a git repo, such as on GitHub.
  • Connect it to GitMovie with a single click.
  • Continuously deploy videos on permalinks, e.g. https://gitmovie.com/AwesomeUser/AwesomeVideo
  • Auto-translate videos into other languages with machine translation, such as DeepL.
  • Serve translations like how you would i18n software docs, e.g. https://gitmovie.com/de/AwesomeUser/AwesomeVideo
  • And many more awesome features!

How would that even work? See some examples at Manim Voiceover website.
